# Extraer la contraseña del WIFI de un dispositivo con MIUI.

A partir de determinada versión de MIUI es posible compartir la contraseña desde el gestor de WIFI. En un dispositivo se muestra un QR con la contraseña y en el otro se usa la aplicación de escanear QR para leer esa contraseña y configurar la red WIFI automátivamente.

Para ver el QR, dejamos pulsado el icono del WIFI:

Pulsamos sobre la red a la que está conectado el movil, si está disponioble esta opción veremos "Toca para compartir la contraseña".

Al pulsar aparecerá un código QR. Tenemos que echarle un pantallazo, esto es posible deslizando 3 dedos sobre la pantalla a la vez.

El pantallazo aparecerá como una notificación en pantalla durante unos segundos, si lo pulsamos podremos manipular la imágen. Si no podremos encontrarla y manipularla desde la galería, aparecerá en "Álbumes > Recientes" y el "Álbumes > Capturas de pantalla".

Recortamos la ímagen hasta solo quedarnos con el QR y algo de espacio a los lados. No es necesario ajustar el QR. Usaremos alguna aplicación web que nos permita subir ficheros desde el dispositivo para decodificar el QR.

Subimos el fichero usando Documentos a la web ZXING. Veremos el "Parsed result" la contraseña entre comillas.
