# Instalando Windows 10 en un portatil viejo (2018)

La idea es limpiar un portatil viejo formateandolo e instalarle una versión actualizada de Windows 10 pero eliminando/bloqueando procesos de recoleccón de datos o procesos que no necesitamos.

El primer paso es recuperar la licencia de Windows de esa máquina, usando la herramienta ProductKey de Nirsoft. De aquí nos basta con echarle una foto a la licencia de producto. Es recomendable en cualquier caso exportar la clave a un fichero y copiarla en algún dispositivo ajeno al portatil que se va a formatear.

Instalaremos el S.O. en una memoría de 16Gb usando la herramienta disponible en Windows 10 Tool.La herramienta se encarga de descargar y preparar la memoría para la instalación.

3.83Gb usados aunquen nos pide una memoría de almenos 8gb.

Para arrancar en la BIOS ahora es necesario arrancar el S.O. que teniamos antes y apagarlo seleccionando la opción de reinicio, entonces durante este reinicio pulsamos F2 de acuerdo a la documentación de ACER. Esto nos permite saltarnos el FastBoot de Windows.

Se instala Windows 10 siguiendo las instrucciones que podemos encontrar en este post de reddit en los apartados "Before/During Installation" y "After Installation". Se describe como realizar una instalación "Personalizada de Windows 10".

A continuación usaremos este script de PowerShell para limpiar la instalación por defecto de W10.

https://github.com/Disassembler0/Win10-Initial-Setup-Script , descargable desde el enlace ZIP.

Si en el equipo antes había windows podemos eliminar versiones antiguas con una propia herramienta del S.O. Para ello ejecutamos el "Liberador de espacio en disco" y pulsamos "Limpiar archivos del sistema".

Esto nos deja con una build 10.0.17134.112 funcional con determinadas características deshabilitadas y la telemetría desactivada. A partir de este punto el S.O. se bajará actualizaciones y es posible que reinstable o habilite algo de lo que hemos eliminado con la herramienta, es posible que haya que ejecutarla de nuevo en el futuro.
